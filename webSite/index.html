Welcome to the Board18 Web Site!

BOARD18 is a tool for playing “18xx style” games via Email.

BOARD18 is used by players of 18xx games to display 18xx game boards and stock market charts.
Track, tiles and tokens can be easily manipulated by the players on these displays.

BOARD18 web sites are run as web applications and are accessed via each player's web browser.

Please use the links to the left to find out more about BOARD18.

And for more information on 18xx style games, you can reference this wikipedia article.
